Welcome!

My name is Kevin Marshall, and I design and build dynamic websites, desktop and mobile applications, and databases by applying analytical and agile methodologies.
I am a graduate of the Conestoga Computer Programmer/Analyst three-year advanced diploma program with Distinction. I love learning new technologies in my free time and find it endlessly satisfying to expand my coding language repertoire. Please do not hesitate to contact me about any opportunities. Thanks!

What are my skills?

C#

Within Visual Studio, I have built C# desktop software using GUIs and ASP.NET-based dynamic web applications, which often included connections to relational databases such as MySQL.

JavaScript

I have used JavaScript to build many dynamic web applications utilizing several different JavaScript technologies and libraries. These include jQuery, Node.JS and Vue.JS, and vanilla JavaScript.

Java

My Java experience includes building mobile applications within Android Studio and dynamic distributed web applications within NetBeans and IntelliJ with connections to relational databases.

Databases

I have designed and implemented various database technologies, including MySQL, Oracle, SQLite, and Access. I have used SQL to perform all C.R.U.D. operations within my dynamic websites to interact with the databases. I also have experience with MonogoDB, a NoSQL database style.

HTML/CSS

Using HTML and CSS, I employ best practices for designing user experience and user interfaces for front-ends with user journeys in mind. I construct pages with logical navigation, clearly displayed information, and responsive feedback to the user to achieve this.

Game Coding/Design

I collaborated with a partner to design and build two short games, in which I was the programmer, gameplay designer, and writer. One of our projects was featured on the itch.io Twitter feed, seen here. Our next project, which we are currently working on, will be built within Unity.
